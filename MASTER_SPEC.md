# SlothOS Master Spec

SlothOS is a Debian stable based distro.

Current active goal:
- Debian stable base.
- SlothOS branding at internal and user-facing levels.
- Built-in Wine.
- Minimal clean implementation.
- ISO as final output.

Kept in scope but not expanded yet:
- sloth admin model.
- --no-install-recommends.
- /Applications and /Games policy.
- Btrfs pooled storage philosophy.
- Kubuntu host + VirtualBox + Debian stable VM.
- future Laptop Edition growth.
